About These Use Cases — The HITL Requirement

Every output shown above is reviewed by the licensed professional before it is acted on. This is not a disclaimer — it is the design. The Human-in-the-Loop step is where the physician verifies that "Patient X was prescribed Drug Y on Date Z" matches what is actually in the chart before signing a prior auth letter. The attorney confirms that "Document 47, page 12" actually says what the summary claims before citing it in a filing.

Cloud AI models generate from parametric memory — they can fabricate citations with high confidence. The Sentinel Node generates from retrieved document chunks only. Every claim is anchored to a specific page in a specific document that exists in your local file system. The professional's job is to verify, not to trust.
